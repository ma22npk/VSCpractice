ヤッホー

  • coomand + Bでアクティブバーの表示と非表示の切り替え
  • 未保存の時はファイル名の横に⚪️がつく command + s でセーブする
  • VSCでは選択なしの coomand + Xで行ごと切り取り、command + C　でコピーができる。
  • VSCでCSSファイルを作成していなくても、link,scriptタグで指定してマウスホバーし、 option + クリックで作成が可能
  • 表示されているファイルのタブを適当な位置にドラッグすることで、画面を分割することができる
  • 選択してcommand + / でコメントアウト化することができる。
  • 選択してoption + command + s で全てのファイルをセーブすることができる。
  • Command + , で設定画面を開く
  • Command + shit + P でコマンドパレットを開く
  • option + 上下 で行の移動ができる
  • option + shit + 上下 で行の 複製 ができる
  • command + enter で 下方向に行の 挿入 ができる
  • shift + command + enter で 上方向 に行の 挿入 ができる
  • command + D で同じ文字列を選択
  • command + shift + L で同じ文字列を 全て 選択
  • command + option + 上方向 で 全て 選択
  • command + Fで検索窓の表示
  • command + Gで次の検索結果,　shift + command + G でもどる
  • command + option + F で検索窓の表示
